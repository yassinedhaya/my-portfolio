# Dev Dungeon Showcase 🎮💻

**Dev Dungeon Showcase** is a unique, interactive, and gamified portfolio designed for software engineers and game developers. It combines professional project presentation with playful, engaging elements to make a memorable impression on potential employers and collaborators.

## About the Project
This portfolio is designed to showcase:
- **Projects:** Interactive demos of software and games.
- **Skills:** Gamified skill bars and achievements.
- **Resume/CV:** Downloadable and visually engaging.
- **Personality:** Fun, interactive elements and easter eggs.

It’s built to stand out from traditional portfolios by blending professional presentation with playful interactions, making it both memorable and enjoyable for visitors.

---

## Features
- Fully **responsive and modern design**
- **Gamified navigation**: mini-games, achievements, and level-up progress
- **Project showcase** with previews, hover effects, and interactive demos
- **Animated skill charts** and progress bars
- **Downloadable CV** presented in a unique game-style layout
- Fun **interactive contact form** with playful animations
- Easter eggs hidden for visitors to discover

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Frameworks & Libraries:** React, Tailwind CSS, Framer Motion, Lucide React icons
- **Deployment Ready:** Fully functional, easy to host
